Project Overview

ElectroHub is an end-to-end Sales Analytics dashboard built using Power BI, designed to analyze sales performance, profitability, discounts, customer behavior, and geographic trends across multiple product categories such as Electronics, Footwear, Clothing, Home Appliances, and more.

The dashboard enables business stakeholders to track KPIs, identify top & underperforming products, and make data-driven decisions using interactive visuals and filters.
